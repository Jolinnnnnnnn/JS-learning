对象新属性申明语句：person.email='jolin@htomail.com'

对象解构赋值，从 person 对象中提取 firstName 和 lastName 属性，并分别赋值给两个新的常量 firstName 和 lastName。 这种语法允许你从对象中提取特定的属性，并直接将它们赋值给新的变量。在这个例子中，firstName 和 lastName 将分别被赋值为 "Joey"，"lin"
// 直接在解构赋值时指定新的值 const { firstName: Joey, lastName: lin } = person;
